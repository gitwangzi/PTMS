Operación de barra de control de navegación
La barra de control de navegación como Fig 8 Barra de control de navegación se muestra.
 
Fig 8 Barra de control de navegación
Las instrucciones de la barra de control de navegación como Table 5 Instrucciones de la barra de control de navegación se muestra .
Table 5 Instrucciones de la barra de control de navegación
Icono
Función
Instrucción de operación
Nombre de usuario
Se muestra el nombre de usuario actual
Estado de servicio
Muestra el estado actual del servicio del sistema y “Conectando” está en el estado normal.
Estado de vehículo en línea
Se muestra el estado en línea del vehículo administrado por el sistema actual El gráfico de barras de porcentaje muestra el porcentaje de vehículos en línea en todos los vehículos (el verde representa en línea; el porcentaje representa la proporción de vehículo en línea), y el número representa el número total de vehículos.
Información de usuario
Haga clic para mostrar la ventana de “Información de usuario”, en que puede cambiar la información básica del usuario actual.
Cambio de contraseña
Haga clic para mostrar la ventana de “Cambiar contraseña”, en que puede modificar la contraseña de inicio de sesión del usuario actual.
Ayuda
Haga clic en los diferentes menús de funciones para ver la documentación de ayuda de esta función.
Papelera de reciclaje
Se registran y restauran los vehículos eliminados, los suites de seguridad y la información del usuario.
Salir del sistema
Puede realizar la función de cierre de sesión del sistema, y debe iniciar sesión nuevamente después de cerrar sesión.
