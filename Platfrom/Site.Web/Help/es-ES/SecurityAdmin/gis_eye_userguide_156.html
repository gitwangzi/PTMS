Gestión de roles
Puede agregar, eliminar, modificar, ver y configurar permisos operativos para cada rol.
Haga clic en Rol y usuario→Rol en la barra del menú de navegación a la izquierda para entrar a la página “Rol”, como se muestra en la siguiente figura.
 
Fig 84 Gestión de roles
•  Agregar rol
1. Hace clic en el botón 〖〗, aparece el cuadro de diálogo “Agregar”.
2. rellene el nombre del rol, seleccione el tipo de rol y agregue una descripción del rol.
3. Haga clic en el botón 〖Guardar〗 para finalizar la adición.
El rol recién agregado se puede ver en la lista de roles.
Note:
–  Cuando agrega el rol, “Nombre del rol” es un campo obligatorio
–  Cada usuario solo puede tener un rol.
–  Después de que se asigna un rol a un usuario, el usuario puede usar el sistema.
•  Modificar rol
1. Haga clic en el botón después del rol 〖〗que desea modificar, aparezca el cuadro de diálogo“Modificar”.
2. Modifique la información del rol y haga clic en〖Guardar〗 para completar la modificación.
Note:
El sistema proporciona roles de forma predeterminada, como SecurityMonitor, SecurityAdmin, MaintainMonitor, MaintainAdmin, y no se puede modificar.
•  Ver detalles del rol
Haga clic en el botón después del rol 〖〗 en la lista de roles para que aparezca el cuadro de diálogo de detalles y ver los detalles del rol.
•  Eliminar rol
Elimine la información de registro perdida que se ha agregado.
1. Haga clic en el botón después del rol 〖〗en la lista de roles, aparezca el cuadro de diálogo de confirmación de eliminación.
2. Hace clic 〖Aceptar〗 para finalizar la operación de eliminación
El rol recién agregado se puede ver en la lista de roles.
Note:
–  El sistema proporciona roles de forma predeterminada, como SecurityMonitor, SecurityAdmin, MaintainMonitor, MaintainAdmin, y no se puede eliminar.
–  La eliminación de un rol requiere que todos los usuarios bajo el rol se eliminen de la relación antes de que puedan eliminarse.
•  Ver rol
Ingrese la palabra clave en “nombre de rol”, 〖〗 el botón y la lista para mostrar el rol consultado.
•  Configurar permisos de roles
1. Haga clic en el botón después del personaje 〖〗 en la lista de roles, aparezca el cuadro de diálogo “Selección de rol”.
2. marque las funciones en la lista de funciones para configurar los permisos de operación para el rol.
3. Haga clic en 〖Guardar〗 para completar la operación de configuración de permisos.
Note:
El sistema proporciona los permisos de rol predeterminado, que solo se pueden ver y no se pueden modificar.
