Distribución de pantalla
La pantalla de servicios se puede dividir en una barra de control de navegación, una barra de servicio de navegación, una barra de menú de navegación y un área de visualización de contenido de acuerdo con el diseño de la interfaz, como se muestra en la figura.
 
Fig 7 Distribución de pantalla
Área 1: barra de control de navegación. Pueda hacer clic en el botón correspondiente para utilizar las funciones relacionadas en cualquier momento durante la operación.
Área 2: barra de servicio de navegación. Incluye siete servicios importantes que son monitoreo en tiempo real, consulta histórica, administración del tráfico, administración de información básica, informes estadísticos, video wall y operaciones relacionadas con la administración del sistema.
Área 3: barra de menú de navegación. Se muestra menú de navegación diferente según el servicio seleccionado por la barra de servicio de navegación del área 2.
Área 4: área de muestra de contenido. Se muestra diferente interfaz de operación según el menú seleccionado por la barra de menú de navegación del área 3.
