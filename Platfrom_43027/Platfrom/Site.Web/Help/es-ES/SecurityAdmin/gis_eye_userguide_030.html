Herramienta de navegación del mapa
La herramienta de navegación del mapa facilita la operación del mapa por usuarios, incluido el desplazamiento hacia el sur, hacia el norte, hacia el este, hacia el oeste, la rotación del mapa, el acercamiento, el alejamiento y las operaciones de pantalla completa.
Cuando mueva el mouse a la herramienta de navegación, el estado de la herramienta de navegación está opcional, como se muestra en la siguiente figura. Ahora pueda realizar operaciones relacionadas en el mapa.

Table 8 La instrucción de herramienta de navegación de mapa
Código
Función
Instrucción de operación
1
Moverse hacia el norte
El mapa muestra que el área se mueve hacia el norte.
2
Moverse hacia el este
El mapa muestra que el área se mueve hacia el este.
3
Moverse hacia el sur
El mapa muestra que el área se mueve hacia el sur.
4
Moverse hacia el oeste
El mapa muestra que el área se mueve hacia el oeste.
5
Acercar
El mapa actual muestra que el área se acerca.
6
Alejar
El mapa actual muestra que el área se aleja.
7
Acerca el control deslizante
Arrastre el control deslizante para acercar o alejar el mapa.
8
Girar
Cuando la herramienta de navegación está en el estado opcional, haga clic en el botón izquierdo del mouse para girarla a realizar la rotación de la dirección del mapa.
